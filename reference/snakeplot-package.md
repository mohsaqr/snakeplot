# snakeplot: Serpentine Plots for Long Timeline, Sequential and Survey Data

Visualize long timelines, extended sequences and temporally chained
survey responses and experience sampling data using intuitive serpentine
(snake) plots. Supports distribution bars, tick-mark plots, inter-item
correlation arcs, faceted multi-construct panels, and daily time-of-day
positioning for ecological momentary assessment data.
